# Syntecxhub_Website-Traffic-Analysis


## Overview
This project analyzes website traffic data using **Python** and **Power BI** to identify traffic trends, user engagement, conversion performance, and page performance. The cleaned data is visualized through charts in Jupyter Notebook and an interactive dashboard in Power BI.

## Tools & Technologies
- Python
- Jupyter Notebook
- Power BI
- Pandas
- Matplotlib
- Seaborn
- OpenPyXL
- Microsoft Excel

## Data Cleaning
- Imported Excel dataset using Pandas.
- Checked and handled missing values.
- Removed duplicate records.
- Converted the Date column to datetime format.
- Verified data types.
- Created new metrics such as Conversion Rate and Goal Completion Rate.

## Visualizations
- Traffic Source Distribution
- Sessions Over Time
- Revenue Trend
- Sessions by Page
- Revenue by Traffic Source
- Bounce Rate by Page
- Goal Completions by Page
- Sessions vs Revenue Scatter Plot
- Correlation Heatmap

## Power BI Dashboard
The dashboard includes:
- KPI Cards (Sessions, Users, Revenue, Conversion Rate)
- Traffic Source Analysis
- User Engagement Metrics
- Revenue Analysis
- Page Performance
- Interactive Filters (Date, Traffic Source, Page)

## Key Insights
- Identified top traffic sources.
- Evaluated user engagement metrics.
- Tracked conversions and goal completions.
- Compared high- and low-performing pages.
- Generated business insights using interactive dashboards.
